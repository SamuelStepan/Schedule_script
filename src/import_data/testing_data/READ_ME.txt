In this file is excel file with score for some matches. First time you run this program, program should export only 2 files, "categories_matches_score.xlsx"
and "categories_matches_data.xlsx". Then you can copy file "categories_matches_data.xlsx" and put it into folder import_data and set your own score for matches
or you can copy this file (if you do not change teams or fields). Then next run files "categories_matches_score.xlsx" and "categories_matches_data.xlsx"
should have updated score in them and there should be also new file "categories_ranking.xlsx".
